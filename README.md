# 🩺 AI-Powered Gait Analysis with Real-time Biofeedback

This project is a specialized tool for **Physical Therapists** to analyze knee kinematics during the gait cycle using Computer Vision.

## 🚀 Key Features
- **Real-time Joint Tracking:** Uses MediaPipe to calculate knee angles with high precision.
- **Visual Biofeedback:** Screen turns red when terminal knee extension is insufficient (< 155° during stance).
- **Auditory Biofeedback:** Immediate "Beep" alert to facilitate motor learning.
- **Data Logging:** Automatically exports every frame's data to a CSV file for clinical reporting.

## 📊 Clinical Application
Used for assessing gait abnormalities and providing immediate corrective feedback for patients during rehabilitation. It helps in documenting patient progress with real numerical data.

## 🛠 Tech Stack
- Python
- OpenCV
- MediaPipe
- Winsound
